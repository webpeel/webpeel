How it works

1

Simple Fetch

Fast HTTP request (~100ms). Works for 90% of sites. No browser overhead.

2

Browser Mode

Headless Playwright (~2-5s). Handles JavaScript rendering and dynamic content.

3

Stealth Mode

Full anti-detection (~5-10s). Bypasses Cloudflare and most protections.
