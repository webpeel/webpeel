HOW IT WORKS

Three lines. Live in minutes.

01

Install

One command. Works everywhere — macOS, Linux, Windows. Zero dependencies beyond Node.

# Install globally npm install -g webpeel
02

Fetch

Send any URL. WebPeel auto-detects what's needed — simple HTTP, browser rendering, or advanced mode.

# Auto-detects: simple → browser → advanced webpeel "https://example.com" --readable
03

Integrate

Clean markdown, metadata, structured data — ready for your agent in milliseconds.

import { peel } from 'webpeel' const page = await peel('stripe.com')
