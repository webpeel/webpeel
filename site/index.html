Frequently Asked Questions
What happens if I exceed my limits? ‚ñº
WebPeel has two tiers: hourly burst limits (hard ‚Äî returns 429) and weekly quotas (soft ‚Äî degrades to HTTP-only mode instead of blocking). Hourly resets automatically; weekly resets every Monday. Free has no overages ‚Äî wait for the reset. Pro/Max have higher limits and can add extra usage credits when needed.
Can I cancel anytime? ‚ñº
Yes, absolutely. No contracts, no commitments. Cancel your subscription anytime from your dashboard.
Do you store the pages I fetch? ‚ñº
No, we never store fetched content. We only log metadata (URL, timestamp, response time) for analytics and debugging.
Is there an enterprise plan? ‚ñº
Yes! Open a GitHub Issue or email support@webpeel.dev for custom pricing and dedicated infrastructure.
Can I self-host instead? ‚ñº
Yes, WebPeel is open source under the AGPL-3.0 license. Install it with npm install -g webpeel and run your own instance.
