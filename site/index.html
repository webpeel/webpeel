Frequently Asked Questions
What happens if I exceed my limits?▼
WebPeel has two tiers: hourly burst limits (hard — returns 429) and weekly quotas (soft — degrades to HTTP-only instead of blocking). Hourly resets automatically; weekly resets every Monday. Free has no overages. Pro/Max can add extra credits.
Can I cancel anytime?▼
Yes, absolutely. No contracts, no commitments. Cancel anytime from your dashboard.
Do you store the pages I fetch?▼
No. We never store fetched content. We only log metadata (URL, timestamp, response time) for analytics and debugging.
How is WebPeel different from Firecrawl?▼
WebPeel has more MCP tools (18 vs ~6), a built-in deep research agent, smart auto-escalation (HTTP → browser → stealth, zero config), YouTube transcript extraction, LLM-free Q&A, domain-aware extractors, 27+ search sources, hotel search, branding extraction, and a more generous free tier. Fully open source under AGPL-3.0. 100% success rate vs Firecrawl's 93.3% in our benchmark.
Is there an enterprise plan?▼
Yes! Open a GitHub Issue or email support@webpeel.dev for custom pricing and dedicated infrastructure.
Can I self-host instead?▼
Yes, WebPeel is open source under AGPL-3.0. Install with npm install -g webpeel and run your own instance.
