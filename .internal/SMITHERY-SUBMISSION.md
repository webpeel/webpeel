# Smithery Submission Guide

## How to Submit

1. Go to https://smithery.ai
2. Sign in with GitHub (JakeLiuMe)
3. Click "Submit Server" or "Add MCP Server"
4. Fill in:
   - **Name:** WebPeel
   - **Repository:** https://github.com/webpeel/webpeel
   - **Description:** Fast web fetcher for AI agents — scrape, search, crawl, map, extract. Smart escalation from HTTP to headless browser to stealth mode. 9 MCP tools. No API key needed.
   - **Category:** Web Scraping / Data Extraction
   - **Tags:** web-scraping, ai-agents, mcp, markdown, crawl, stealth, search

## smithery.yaml Already Created

The `smithery.yaml` file is in the repo root. It tells Smithery how to start the MCP server.

## After Submission

The server will be installable via:
```bash
npx @smithery/cli install @webpeel/webpeel
```
